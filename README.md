# Restaurante Persépolis - a refactored restaurant website for a Persian restaurant in València.

#### Preview

 - Open `index.html` in your browser or use a local web server to preview the site.

## Getting Started

1. Open the folder in your editor.
2. Open `index.html` in your browser, or serve the directory with a local web server.

## Notes
```
This repository contains the refactored Restaurante Persépolis website. The original template has been rebranded for a Persian restaurant in València.
```
